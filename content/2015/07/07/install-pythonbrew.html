鉴于python各版本的不兼容性，一直想安装一个python的版本管理工具。今天终于抽出空来实践了一把。

我安装的是pythonbrew。

安装

curl -kL http://xrl.us/pythonbrewinstall | bash

配置

在~/.bash_profile中添加：

[[ -s $HOME/.pythonbrew/etc/bashrc ]] && source $HOME/.pythonbrew/etc/bashrc

更新

pythonbrew update

类库

我是通过homebrew安装openssl的，但安装完并没有软链接，导致之后安装的python并不成功，所以要先调整一下。

具体方法是执行一下以下命令：

brew link openssl

其它有相似情况的类库也照些处理。

安装python

pythonbrew install 2.7.10
pythonbrew install 3.4.3

转换当前python环境

pythonbrew switch 2.7.10

列出已安装的python环境

pythonbrew list

卸载python

pythonbrew uninstall 2.7.10

清除下载的源文件及编译生成的临时文档

pythonbrew cleanup
